Obtaining the psiKeds Repository

You can fetch the psiKeds repository by using the standard github commands

  • git clone https://github.com/dtag-dbu/psikeds.git
  • git clone git@github.com:dtag-dbu/psikeds.git

By fetching the master branch, you receive the latest stable release. In combination with the installed and configured Maven system you can directly build, test, and install psiKeds by using the corresponding mvn commands.

Obtaining the psiKeds tarballs 

  • Currently, you can also download the psiKeds sources as zip archive by calling the github url https://github.com/dtag-dbu/oslic/zipball/master
  • Later on you will probably also be able to download the psiKeds sources as a tarball.
  • And later on you will probably also be able to download the psiKeds jar file.
