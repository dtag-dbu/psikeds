Preparing the System Perequisites

The most simple environment should be a successfully running a GNU-Linux system (like Ubuntu 14.04) offering

  • Bash
  • OpenJDK 7 or later
  • Maven 3 or later
  • Git 1.7 or later
  • Eclipse Juno or later
  • Tomcat 7 or later
  • Firefox, Chrome, Opera,

But you could also use an Apple system or a Window system offering the corresponding components.
